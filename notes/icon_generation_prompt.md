# TaskFlow icon prompt

Use this prompt in ChatGPT or another image model to generate the app branding set for `TaskFlow`.

## Main prompt

Create a clean, modern mobile app icon for an app called `TaskFlow`.

Brand direction:
- productivity, calm focus, task organization
- mobile-first, friendly, minimal, polished
- should feel trustworthy and practical, not childish
- visual style: premium startup utility app

Graphic idea:
- a rounded square app icon
- central symbol combining:
  - a flowing checkmark or path
  - and a subtle task/list structure
- the mark should suggest: "capture quickly, organize smoothly"

Color direction:
- primary color family: deep indigo / electric violet-blue
- support colors: soft periwinkle, white, very light lavender-gray
- high contrast, clean gradients allowed
- avoid too many colors

Constraints:
- no text inside the icon
- no photorealism
- no 3D mockup device frames
- simple enough to remain recognizable at small size
- balanced for Android and iOS
- crisp edges, centered composition, generous padding

Deliverables to design from the same visual system:
1. `icon.png`:
   - square app icon
   - 1024x1024
   - transparent-free solid background preferred

2. `splash-icon.png`:
   - same brand mark
   - more centered and spacious
   - optimized for use on a splash screen
   - 1024x1024

3. `favicon.png`:
   - simplified version of the same mark
   - highly legible at tiny size
   - 256x256

Quality bar:
- elegant
- minimal
- iconic
- consistent across all three assets

## Optional alternative prompt

Create 3 consistent icon concepts for `TaskFlow`:
- concept A: flowing checkmark
- concept B: checklist + motion path
- concept C: abstract time-block / planner tile mark

Keep all concepts in the same color family and style.
